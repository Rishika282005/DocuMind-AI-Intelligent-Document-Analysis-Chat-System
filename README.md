# 📚 DocuMind AI

An AI-powered document analysis system that allows users to upload documents, generate summaries, chat with documents, perform semantic search, and extract important insights.

---

## 🚀 Features

- 📄 Upload PDF, DOCX and TXT files
- 🤖 AI Document Summary
- 💬 Chat with uploaded documents
- 🔍 Semantic Search
- 🧠 Document Insights
- 📊 Performance Metrics
- 🌙 Dark / Light Mode
- 🎨 Responsive Dashboard

---

## 📊 Performance Metrics

- Word Count
- Character Count
- Page Count
- Reading Time
- File Size
- Number of Chunks

---

## 🧠 Document Insights

- Key Topics
- Emails
- Dates

## ⚙ Installation

### Clone

```bash
git clone https://github.com/YOUR_USERNAME/DocuMind-AI-Intelligent-Document-Analysis-Chat-System.git
```

---

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn app:app --reload
```

---

### Frontend

```bash
cd frontend

npm install

npm run dev
```
